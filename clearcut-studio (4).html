🏢 Team Efficiency Guide — 30 Members

Best practices for using ClearCut Studio at KPMG scale

⚡ Workflow — Process Images Fast

Batch upload Drop up to 30 images at once per session — AI runs sequentially while you work on previous results
Preset canvas Agree on one canvas size across the team (e.g. 1920×1080 or LinkedIn 1200×628) and use the preset every time
Start AI first Upload all images before touching any tools — AI queues them all and processes while you refine

👥 Team Split Strategy

10 uploaders Upload and batch-process images, download clean PNGs, pass to refiners
15 refiners Open downloaded PNGs, use Restore/Erase/Hair brush for edge cleanup
5 finalizers Apply background, adjust scale/position, final export

💾 File Naming Convention

Input format LASTNAME_FIRSTNAME_role.jpg
Output suffix App auto-adds _clearcut.png — keep this for traceability
ZIP export Use "Download All as ZIP" to batch export — saves ~2 minutes per person

🎯 Quality Tips

Image input Use the highest resolution source available — app preserves full native resolution
Feather = 2–4px Ideal for headshots on solid/gradient backgrounds. Use 0 for print
Auto Clean first Run "Auto Clean" before manual brush work — fixes 80% of edges automatically
Hair Refine (H) Use on flyaway hair before downloading — biggest quality difference for headshots
No server needed AI runs 100% in browser — each user can run independently, no bottleneck

📋 KPMG Background Presets

LinkedIn headshot White (#FFFFFF) background, 1200×628 canvas
Internal directory KPMG Blue (#00338D) or white background, Square 1080×1080
Presentations Transparent PNG, import directly into PowerPoint
